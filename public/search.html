Search through the API by Page Number

You can get a speciific post by Title of the post or by the Author Name and by both title and the author name

Example usage

http://localhost:3000/?page=2

http://localhost:3000/?author=Ibrahim

http://localhost:3000/posts/?title=ExamplePost

On Error, it will return this:
